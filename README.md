# Fuzzy Lookup Portal

A browser-based tool for fuzzy matching between two lists. Upload or paste your data and instantly find similar items across lists.

## Features

- **Upload or Paste** - Drag and drop CSV/TXT files or paste data directly
- **Adjustable Threshold** - Control how strict the matching is (0-100%)
- **Multiple Match Strategies** - Levenshtein distance, bigram similarity, Jaccard index, and substring matching
- **Export Results** - Download matches as CSV
- **Privacy First** - All processing happens in your browser. No data is sent anywhere.

## Usage

1. Visit the live portal
2. Add items to **List A** (source) and **List B** (lookup)
3. Adjust the match threshold and max results
4. Click **Find Matches**
5. Export results as CSV if needed

## Hosting on GitHub Pages

1. Go to your repository Settings
2. Navigate to Pages
3. Set Source to "Deploy from a branch"
4. Select the `main` branch and `/ (root)` folder
5. Click Save
